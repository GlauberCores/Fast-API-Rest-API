# FastAPI REST API - Orders System

Projeto de API REST desenvolvido com FastAPI para gerenciamento de usuários e pedidos.

## Tecnologias utilizadas

- Python
- FastAPI
- SQLAlchemy
- SQLite
- Uvicorn

## Funcionalidades

- Cadastro de usuários
- Autenticação básica
- Criação de pedidos
- Listagem de pedidos
- Atualização de status de pedidos
